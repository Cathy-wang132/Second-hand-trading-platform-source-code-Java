# Second-hand-trading-platform-source-code-Java
### 二手交易平台源码-Java
#### 1、介绍

这是一个完整的二手交易平台项目，包含平台管理端和手机H5用户端，通过本项目，可以掌握完整的前后端开发流程，学习常见的开发框架和工具，提升技术水平！后端：Java 21 + Spring Boot 2.7 框架，数据库使用 MySQL 8.0，构建工具采用 Maven 3.9，前端：管理端使用 Vue 2 + Element UI，用户端使用 Uni-app！

#### 3、系统功能

（1）平台管理端

· 用户管理

· 商品分类管理

· 二手商品管理

· 订单管理

· 管理员管理

· 资讯管理

（2）用户端

· 二手商品展示

· 分类筛选

· 商品详情

· 订单凭证与我的订单

· 联系卖家

· 资讯查看

· 我的发布

注： 用户端使用 Uni-app 框架开发，支持多端打包，理论上可以适配小程序、APP 等多个平台，但本项目目前仅适配了 H5 版本，其他平台需根据实际需求自行调整和适配。

#### 4、技术栈与工具

· 后端：Java 21 + Spring Boot 2.7 框架，数据库使用 MySQL 8.0，构建工具采用 Maven 3.9

· 前端：管理端使用 Vue 2 + Element UI，用户端使用 Uni-app

· 开发工具：管理端使用 VSCode 编辑器，用户端使用 HBuilder X 编辑器

· 环境要求：·前端 Node.js 版本 16.20 以上，Java 21 以上，Spring Boot 2.7，MySQL 8.0，Maven 3.9

#### 5、项目亮点

· 功能全面： 从用户管理、商品管理到订单和资讯管理，涵盖了二手交易平台的常见功能，适合初学者进行实战开发。

· 高效的开发框架： 使用 Spring Boot 作为后端框架，开发效率高，性能优越，帮助你提升 Java 后端开发技能。

· 完整的前后端技术栈： 结合 Java、Spring Boot 和 MySQL 等后端技术，提供全栈应用开发经验。

· 易于扩展： 项目结构清晰，便于二次开发和功能扩展，适合进一步的自定义和优化。

· 适配多端： 用户端使用 Uni-app 开发，虽然目前适配了 H5，但你可以根据需求扩展到小程序或原生 APP。
